# 🇮🇳 Atmanirbhar Yatra

### Swadeshi Travel & Tourism Platform for Local Empowerment
Atmanirbhar Yatra is a smart travel platform designed to promote India’s local tourism ecosystem by connecting travelers with local hotels, homestays, guides, transport providers, and cultural experiences. The platform supports small businesses and encourages sustainable, Swadeshi tourism.

### 🚀 Features
🔐 User Authentication (Login/Signup)

🏨 Browse Hotels & Homestays

🧭 Discover Local Tour Guides

🚗 Transport & Travel Services

🔍 Search & Explore Destinations

📅 Booking System (In Progress)

⭐ Review & Rating System (Upcoming)

### 🏗️ System Architecture

The platform follows a client-server architecture:

* Client Layer – Web interface for users and providers
* Application Layer – Handles authentication, services, booking
* Backend API – Processes requests
* Database Layer – Stores users, services, bookings

### ⚙️ Installation & Setup
#### Clone the repository
```git clone https://github.com/Aditya-Sachan-Git/Atmanirbhar-Yatra.git```

#### Navigate to project folder
```cd Atmanirbhar-Yatra```

#### Install dependencies
```pip install -r requirements.txt```

#### Run the application
```python app.py```
